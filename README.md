# Fluide

**Fluide** est un blog personnel dédié au bégaiement, à la respiration, à la méditation, et au bien-être en général. Le site propose des articles, des conseils, et des ressources pour améliorer sa qualité de vie et réduire le stress.

Ce projet me permet également de pratiquer le développement web en travaillant sur un sujet qui me touche personnellement.

## 🛠 Installation

1. Cloner le projet :
   git clone https://github.com/ThreeFrogs511/fluide.git

2. Placer le dossier dans `htdocs` (XAMPP)

3. Démarrer Apache via le panneau de contrôle XAMPP

4. Accéder au site à l'adresse : http://localhost/fluide

---

## 🗃️ Base de données

Le projet utilise une base de données MySQL.

Un fichier `fluide.sql` est fourni dans le dossier `/database/`.  
Vous pouvez l'importer via phpMyAdmin ou tout autre outil MySQL pour recréer la structure et les données.

**Nom de la base suggéré :** `fluide_et_libre`
